Local photo organization for Windows

Photo Manager Pro

A desktop workflow for sorting, indexing, reviewing, and cleaning up photo libraries without sending files to the cloud.

Latest release Source code

What it does

Built around practical library maintenance

Sync and sort

Copies media into year folders using EXIF, filename dates, and filesystem timestamps as fallbacks.

Index locally

Stores file metadata, sync events, blur scores, captions, tags, and review decisions in SQLite.

Review safely

Queues duplicates and low-quality candidates before moving files to the recycle bin.

Run in the background

Watches folders, supports startup options, and keeps a single app instance active in the tray.
